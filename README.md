# Snoway Surfers

## About
Snoway Surfers is an endless runner game inspired by Subway Surfers. Avoid Glowing Fences to survive!

## Link to Play online
Made for PC or Mac

https://ilovepino56.itch.io/cold

## Controls
Use 'A' to move left and 'D' to move right.
If you hit a fence, the game starts over.
